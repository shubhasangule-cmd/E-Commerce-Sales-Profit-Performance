# E-Commerce Sales & Profit Performance Analysis | SQL + Tableau

## Project Overview
This project analyzes e-commerce sales data to identify key business insights such as sales trends, regional performance, category profitability, and top-performing products. The analysis was conducted using SQL, and an interactive dashboard was created in Tableau to visualize the results.


## Business Problem
Businesses need to understand:
- Which products generate the highest sales
- Which regions perform best
- Which categories generate the highest profit
- Monthly sales and profit trends

 ## Dataset
The dataset contains the following columns:
- Order Date
- Product Name
- Category
- Region
- Quantity
- Sales
- Profit

## Tools Used
- SQL (Data Cleaning & Analysis)
- Tableau (Dashboard Visualization)
- GitHub (Project Documentation)
